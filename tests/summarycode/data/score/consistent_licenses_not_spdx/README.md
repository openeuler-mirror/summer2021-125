# @invisionag/eslint-config-ivx

# Usage

Install the peer dependencies as development dependencies:
- `eslint^3.19.0`
- `prettier^1.5.2`

Install `@invisionag/eslint-config-ivx` as a development dependency.

In your `.eslintrc.js`:
```js
module.exports = {
  extends: '@invisionag/ivx',
};
```
